    # 一 . 数据类型

    # 1. JavaScript有哪些数据类型

    1.） 简单数据类型

    • 布尔值（Boolean） true和false
    • null 表明 null 值的特殊关键字
    • undefined 表示变量未赋值时的属性
    • 数字（Nmber）
    • 字符串（String）
    • 任意精度的整数 (BigInt) ，可以安全地存储和操作大整数，甚至可以超过数字的安全整数限制。
    • Symbol 一种实例是唯一且不可改变的数据类型。

    2.） 复杂数据类型

    • 数组 （Array）
    • 对象 （Object）
    • 函数 （Function）

    # 2. 简单数据类型没有属性和方法，为什么可以调用某些方法？

    👇

    const str = 'abc'
console.log(str.length)

    为了方便操作原始值，ECMAScript 提供了 3 种特殊的引用类型：Boolean、Number 和 String。这三个基本包装类型允许简单类型直接调用对应的方法。

    过程：👇1.创建一个String类型的实例 => 2. 调用实例上的特定方法 => 3. 销毁实例

    const str = new String('abc')
str.length
str = null

    # 6. 赋值，深拷贝，浅拷贝

    赋值：将某个数值赋值给某个变量。2种情况 👇

    1. 值类型：变量在存储简单数据类型的时候，存储的是值本身
    2. 引用类型：变量在存储复杂数据类型的时候，存储的是地址

    深拷贝：将一个对象从内存中完整的拷贝一份出来，从堆内存中开辟一块新的区域存放新对象，且修改新对象不会影响原对象。 简易实现方式：（不考虑循环引用问题）

    const deepClone = (obj) => {
  if (typeof obj !== 'object') return
  let newObj = obj instanceof Array ? [] : {}
  for (var key in obj) {
    if (obj.hasOwnProperty(key)) {
      newObj[key] = typeof obj[key] === "object" ? deepClone(obj[key]) : obj[key]
    }
  }
  return newObj
}

    浅拷贝：创建一个新对象，赋值对象属性，如果属性值是基本类型，拷贝的是基本类型的值；如果是引用类型，只会赋值地址，这就不能保证两个对象完全独立. 实现：

    const shallowLClone = (obj) => {
  if (typeof obj !== 'object') return
  let newObj = obj instanceof Array ? [] : {}
  for (var key in obj) {
    if (obj.hasOwnProperty(key)) {
      newObj[key] = obj[key]
    }
  }
  return newObj
}

    三者对比

    //赋值
var obj = {
    name: "zs",
    friends: {
        age: 12
    }
}
var obj1 = obj
obj1.name = 'ls'
obj1.friends.age = 11
console.log(obj) //{name: 'ls',friends: {age: 11}}
console.log(obj1) //{name: 'ls',friends: {age: 11}}

    //浅拷贝 
var obj = {
    name: "zs",
    friends: {
        age: 12
    }
}
var obj1 = shallowLClone(obj)
obj1.name = 'ls'
obj1.friends.age = 11
console.log(obj) // {name: 'zs', friends: {age:11}}
console.log(obj1) //{name: 'ls',friends: {age: 11}}

    // 深拷贝 👇
var obj = {
    name: "zs",
    friends: {
        age: 12
    }
}
var obj1 = deepClone(obj)
obj1.name = 'ls'
obj1.friends.age = 11
console.log(obj) // {name: 'zs', friends: {age:12}}
console.log(obj1) //{name: 'ls',friends: {age: 11}}

    总结

    是否指向同一对象 改变基本类型数据 （对比name） 改变引用类型数据 (对比age)
    赋值 是 会使原数据一起改变 会使原数据一起改变
    浅拷贝 否 不会使原数据一起改变 会使原数据一起改变
    深拷贝 否 不会使原数据一起改变 不会使原数据一起改变
