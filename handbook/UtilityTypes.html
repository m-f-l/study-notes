    # 可选类型 Partial<Type>

    • 概念：设置属性都为可选的类型
    • 用法：
      interface Todo {
    title: string;
    descrption: string;
  }

  const todo1: Partial<Todo> = {
    title: '标题'
  }

  const todo2: Partial<Todo> = {
    title: '标题',
    descrption: '描述'
  }
    • 实现
      type MyPartial<T> = {
    [P in keyof T]?: T[P]
  }
  const todo3: MyPartial<Todo> = {
    title: '标题',
//  descrption: '描述',
  }
    • 涉及到的知识点： 关键字 in ,in关键字可以用来遍历联合类型
     type K = "a" | "b"
 type B = {[P in K]: number} => 相当于 type B = {a: number; b: number}

    关键字 keyof 可以用来获取一个类型的键值，并且返回一个联合类型

      interface Todo {
    title: string;
    descrption: string;
  }
  type UnionType = keyof Todo // 相当于  type UnionType = "title" | "descrption"

    # 必选类型 Required<Type>

    • 概念： 设置所有属性都为必选类型，和上面说的Partial相反
    • 用法：
    interface Props {
  a?:number;
  b?:number;
}
const obj: Props ={a: 5}
const obj2: Required<Props> = {a: 5} //Property 'b' is missing in type '{ a: number; }' but required in type 'Required<Props>'.
    • 实现 type MyRequired<T> = {[P in keyof T]: T[P]}

    # 可读类型 Readonly<Type>

    • 概念：设置所有属性都为可读的类型，不能重新修改
    • 用法：
      interface Todo {
    title: string;
  }
  const todo: Readonly<Todo> = {
    title: 'title'
  }
  todo.title = 'HELLO' // Cannot assign to 'title' because it is a read-only property.
    • 实现 type MyReadonly<T> = {readonly [P in keyof T]: T[P]}
    1. Record<Keys, Type>
    • 概念：用于构造一个对象类型，所有的键都是Keys 类型，所有的值都是Type 类型，可以用于映射一个类型的属性到另一个类型
    • 用法：
    interface CatInfo {
  age: number;
  breed: string;
}

type CatName = "miffy" | "boris" | "mordred";

const cats: Record<CatName, CatInfo> = {
  miffy: { age: 10, breed: "Persian" },
  boris: { age: 5, breed: "Maine Coon" },
  mordred: { age: 16, breed: "British Shorthair" },
};

    • 实现 type MyRecord<K extends keyof any ,T> = { [P in K]: T }

    • 实践 如果定义一个对象，有height,weight,age 这三个属性，值都为数字，可以这样写

    type Keys = 'height' | 'weight' | 'age'
const obj: Record<Keys, number> = {
  height: 168,
  weight: 120,
  age: 18
}

或者： const obj: {[key: string]: number}

    # 提取一些属性生成新类型Pick<Type, Keys>

    • 概念： 从Type中提取一些属性（Keys）生成一个新的类型
    • 用法：
    interface Todo {
  title: string;
  descrption: string;
  completed: boolean;
}

type TodoPreview = Pick<Todo, "title" | "completed">;

const todo: TodoPreview = {
  title: 'clean room',
  completed: false
}

type TodoPreview = Pick<Todo, "title" | "completed">; 相当于 type TodoPreview = {title: string;
completed: boolean;
}

* 实现：
type MyPick<T, K extends keyof T> = {
  [P in K]: TP
}

    keyof T 是T类型的键集 K extends keyof T 泛型约束K 就上面这个例子，keyof T 为 "title" | "description" | "completed", 那么K extends keyof T 表示extends用来约束K, K可以是"title" | "description" 或者 "title"或者 "title" | "description" | "completed" ，不能出现除了keyof T 意外的值
